# 3DS-KurtScan
An Attempt to use the 3DS as a (photogammetry) 3D Scanner

#3DS Based Logic:
 1. Snap Photo
 2. Play Tone (Trigger offboard arduino through audio port)
 3. Wait for rotation (defined at startup prompt)
 4. repeat defined amount of times
